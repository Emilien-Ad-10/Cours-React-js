Chapitre 1 : Introduction

React est une librairie JavaScript pour créer des interfaces utilisateurs. Une librairie est là pour résoudre un problème. React n'est pas le meilleur des frameworks, il est juste différents.Il est plus simple que d'autres dans le sens ou il y a assez peu de méthode/fonction à connaitre

Concernant son écosystème, on a différents adapteurs:

  • React DOM pour effectuer des modifications sur le DOM,
  • React native pour la conception d'application native,
  • React desktop pour des applications desktop,
  • etc...

React dispose également d'une syntaxe spécifique appelée le JSX

Chapitre 2 : Premiers pas avec React

L'utilisation des scripts suivants permettent de commencer de la manière la plus simple pour l'apprentissage de React sans avoir à utiliser différents outils comme node.js, etc...

  • <script crossorigin src="https://unpkg.com/react@16/umd/react.development.js"></script>
  • <script crossorigin src="https://unpkg.com/react-dom@16/umd/react-dom.development.js"></script>

Utilisation:

  • React permet la création d'élement,
  • React DOM permet de brancher les différents éléments au DOM (page web)

Cas pratique :

  1. Création d'un paragraphe puis insérer du texte: (Voir avec l'inspecteur d'élément pour la création de balise HTML)
  2. Rajout d'un compte à rebours dans une balise <span>
Remarque :

React est capable de détecter la ou la changement à lieu et permet d'éviter le rechargement complet de la fonction. Pour faire simple, React permet de cibler la ou l'action est requise.
